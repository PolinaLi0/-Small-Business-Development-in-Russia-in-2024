# -Small-Business-Development-in-Russia-in-2024
Проект посвящён анализу состояния малого бизнеса в России в 2024 году на основе открытых данных. В ходе исследования рассматриваются количественная динамика, структура сектора (микро-, малые и средние предприятия), а также особенности предпринимательской активности.
